Demo UPI Mobile Application


The Unified Payments Interface (UPI) has revolutionized the way we transfer money in India. In this project, a simple mock UPI app has been developed, which allows registered users to send money to each other. The app is designed to be user-friendly and easy to use. The user can register by providing their mobile number and verifying it using an OTP. Once registered, the user can add their bank account details to the app. To send money, the user simply needs to select the recipient from their contacts list and enter the amount to be transferred. The app will then authenticate the transaction using the user's UPI PIN and initiate the transfer.
